INSTALL

1
Download SpankTheAgent.zip and unzip it.
2
Drag SpankTheAgent.app to /Applications.
3
macOS blocks unsigned apps. Open Terminal and run once:
xattr -dr com.apple.quarantine /Applications/SpankTheAgent.app click to copy · standard for open-source apps · one time only
4
Open SpankTheAgent — a hand icon appears in the menu bar.
5
Click → Start. Enter password once for accelerometer access.
6
Slap your MacBook. Done.
1
Clone and build:
git clone https://github.com/zkblk/spank-the-agent && cd spank-the-agent && cd app && bash build.sh
2
Or install CLI only (no app):
go install github.com/zkblk/spank-the-agent@latest
3
Run the CLI:
sudo spank-the-agent --agent --warcraft
4
Or drag the built app/SpankTheAgent.app to /Applications — no quarantine needed since you built it locally.
Requires: Go 1.22+ · Xcode Command Line Tools · Apple Silicon Mac

UNINSTALL

1
Quit the app (menu bar → Quit), then remove it:
sudo rm -rf /Applications/SpankTheAgent.app
2
Remove the passwordless sudo rule:
sudo rm /etc/sudoers.d/spank-the-agent
3
Kill any running helper process:
sudo pkill -f spank-the-agent-helper
!
If you ever installed the LaunchDaemon from old docs, remove it first or the helper will keep respawning:
sudo launchctl unload /Library/LaunchDaemons/com.zkblk.spank-the-agent.plist && sudo rm /Library/LaunchDaemons/com.zkblk.spank-the-agent.plist
