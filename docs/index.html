Install

I
Download SpankTheAgent.zip and unzip it.
II
Drag SpankTheAgent.app to /Applications.
III
macOS blocks unsigned apps on first launch. Open Terminal and run once:
xattr -dr com.apple.quarantine /Applications/SpankTheAgent.app click to copy · standard for open-source apps not in the App Store · one time only
IV
Open SpankTheAgent — a hand icon appears in the menu bar.
V
Click → Start. Enter password once for accelerometer access.
VI
Slap your MacBook. Done.
I
Clone and build the app:
git clone https://github.com/zkblk/spank-the-agent && cd spank-the-agent/app && bash build.sh
II
Or install CLI only (no app):
go install github.com/zkblk/spank-the-agent@latest
III
Run:
sudo spank-the-agent --agent --warcraft
IV
Built locally — no quarantine step needed.
Drag app/SpankTheAgent.app to /Applications and open.
Requires: Go 1.22+ · Xcode Command Line Tools · Apple Silicon Mac
